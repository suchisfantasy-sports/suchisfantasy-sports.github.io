# Such Is Fantasy — Style Guide

## Colour Palette (Synthetic Neon)

| Token | Hex | Usage |
|---|---|---|
| Primary Dark | `#18181B` | Background, cards |
| Secondary Dark | `#1f1f22` | Card backgrounds, sections |
| Tertiary Dark | `#27272a` | Borders, dividers |
| Neon Cyan | `#00F0FF` | Primary accent, CTAs, highlights |
| Magenta | `#FF003C` | Secondary accent, alerts, urgency |
| White | `#FFFFFF` | Headings, primary text |
| White 70% | `rgba(255,255,255,0.7)` | Body text |
| White 50% | `rgba(255,255,255,0.5)` | Secondary text, labels |

## Typography
- **Display / Headings:** Clash Display (sans-serif) — bold, uppercase for impact
- **Body / Data:** JetBrains Mono (monospace) — reinforces technical, data-driven feel
- **Scale:** Responsive clamp() values for fluid sizing

## Design System
- **Aesthetic:** Synthetic Neon — dark theme, neon accents, glassmorphism, monospace grid
- **Borders:** 1px solid lines, geometric separators
- **Effects:** Radial gradient glows (cyan/magenta), SVG noise texture overlay, backdrop blur on cards
- **Animations:** GSAP power3.out entrances, staggered reveals on scroll
- **Buttons:** Cyan solid for primary CTAs, ghost outline for secondary, scale-[0.97] on active

## Writing Conventions

### Capitalisation
- **Headlines:** ALL CAPS for hero/section headers (matches Clash Display impact)
- **Subheadings:** Sentence case
- **Brand name:** Such Is Fantasy (capitalised, never "such is fantasy" or "SIF")
- **Sport names:** NRL, EPL, NBA, NFL (always caps)
- **Platform names:** SuperCoach, DraftStars, DraftKings, FPL (match official casing)

### Numbers & Stats
- Always use digits: 73% captain accuracy, 5K+ members, 4 sports
- Use K for thousands: 5K+, not 5,000+
- Percentages always with % symbol
- Money: $9.99/mo, $24.99/mo (always show /mo)

### Formatting
- Short paragraphs (2-3 sentences max)
- Bullet points for lists, features, and comparisons
- Bold key stats and differentiators
- Use sport emoji sparingly in headers: 🏉 NRL ⚽ EPL 🏀 NBA 🏈 NFL

### Tone in Copy
- Punchy, clipped sentences for headlines
- No fluff words: "leverage", "innovative", "cutting-edge", "passionate"
- Australian spelling: colour, favourite, analyse, organisation
- Casual contractions: "you'll", "we're", "don't" — not formal
