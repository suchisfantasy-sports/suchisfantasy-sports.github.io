# Such Is Fantasy — Whop Setup Guide

## Step-by-Step Setup Order

1. Go to whop.com → Create Whop
2. Name: **Such Is Fantasy**
3. Handle: `suchisfantasy`
4. Category: Sports / Fantasy Sports
5. Set up membership tiers (Section 2 below)
6. Build sections + channels using the copy in Section 3
7. Upload your first file (cheat sheet) to the Files app
8. Turn on the Forum app for #team-advice
9. Set visibility rules: free channels visible to all, paid channels gated

## Section 5 — Whop Settings Checklist

- [ ] Profile photo: Such Is Fantasy logo
- [ ] Banner image: NRL themed, bold
- [ ] Public page: ON (so people can discover you)
- [ ] Affiliate program: ON (let members earn by referring — 20–30% commission)
- [ ] Founding member promo: set expiry date for urgency
- [ ] Welcome message: auto-sent when someone joins (use announcement copy)
- [ ] Stripe connected: required before any payments go through
- [ ] Annual pricing: set up alongside monthly for both tiers
- [ ] Social links: TikTok, podcast, Instagram in your Whop profile

## Section 6 — Launch Sequence (What to Post First)

**Day 1 (launch day):**
1. Post welcome message in #announcements
2. Post rules in #rules
3. Post intro prompt in #introductions
4. Drop cheat sheet in #weekly-cheat-sheet (even if rough)
5. Go live on TikTok or post a TikTok driving people to your Whop link

**Day 2–3:**
- Post your first trade targets in #trade-targets
- Drop an injury alert (even if nothing major)
- Respond to every single intro posted

**Week 1 goal:** 30 free members, 10 paying (either tier)
